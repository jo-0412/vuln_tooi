# U-13 Use Secure Password Hashing Algorithm

## 1. Overview

- **Item Code**: U-13
- **Item Name**: Use Secure Password Hashing Algorithm
- **Severity**: High
- **Category**: UNIX > Account Management

This item checks whether account passwords are protected with secure hashing algorithms.

---

## 2. Purpose

The purpose of this check is to reduce the risk of password cracking
even if password hashes are leaked.

Weak algorithms such as MD5 or DES can be cracked more easily,
which may lead to account compromise.

---

## 3. Files to Check

- `/etc/shadow`
- `/etc/passwd`
- `/etc/login.defs`
- Red Hat based systems: `/etc/pam.d/system-auth`
- Debian based systems: `/etc/pam.d/common-password`

---

## 4. Commands to Check

```bash
sudo awk -F: '{print $2}' /etc/shadow | head
grep ENCRYPT_METHOD /etc/login.defs
grep pam_unix.so /etc/pam.d/system-auth
grep pam_unix.so /etc/pam.d/common-password
