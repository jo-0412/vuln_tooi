# U-63 Manage sudoers File Access

## 1. Overview

- **Item Code**: U-63
- **Item Name**: Manage sudoers File Access
- **Severity**: High
- **Category**: UNIX > Privilege Management

This item checks whether `/etc/sudoers` is protected with proper ownership and permissions.

---

## 2. Purpose

The purpose of this check is to prevent unauthorized users from modifying sudo policy
and obtaining administrative privileges.

The `/etc/sudoers` file controls which users or groups can execute commands
with elevated privileges. If this file is not properly protected,
an attacker may modify sudo rules and gain root access.

---

## 3. Files to Check

- `/etc/sudoers`

Optional supporting path:

- `/etc/sudoers.d`

---

## 4. Commands to Check

```bash
ls -l /etc/sudoers
ls -ld /etc/sudoers.d 2>/dev/null
grep -E '^#includedir|^@includedir|^#include|^@include' /etc/sudoers
