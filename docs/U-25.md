# U-25 Check World Writable Files

## 1. Overview

- **Item Code**: U-25
- **Item Name**: Check World Writable Files
- **Severity**: High
- **Category**: UNIX > File System Security

---

## 2. Purpose

To prevent unauthorized modification, deletion, or malicious code injection
through files that are writable by all users.

---

## 3. Target

Entire filesystem

---

## 4. Command

```bash
find / -type f -perm -2 -exec ls -l {} \;
