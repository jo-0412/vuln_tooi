# U-30 Manage UMASK Setting

## 1. Overview

- **Item Code**: U-30
- **Item Name**: Manage UMASK Setting
- **Severity**: Medium
- **Category**: UNIX > File and Directory Management

This item checks whether the system-wide UMASK value is configured securely.

---

## 2. Purpose

The purpose of this check is to ensure that newly created files and directories
are not created with excessive permissions.

If the UMASK value is too weak, newly created files or directories may be exposed
to unauthorized users.

---

## 3. Files to Check

- `/etc/profile`
- `/etc/login.defs`

---

## 4. Commands to Check

```bash
grep -i umask /etc/profile
grep -i '^UMASK' /etc/login.defs
umask
