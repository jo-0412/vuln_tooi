# U-18 /etc/shadow 파일 소유자 및 권한 설정

## 1. 개요

- **항목 코드**: U-18
- **항목명**: /etc/shadow 파일 소유자 및 권한 설정
- **중요도**: 상
- **분류**: UNIX > 계정 관리

이 항목은 비밀번호 해시 저장 파일의 소유자와 권한이 적절히 제한되어 있는지 점검하는 항목이다.

---

## 2. 왜 이 항목이 필요한가

`/etc/shadow` 파일에는 계정의 비밀번호 해시 정보가 저장된다.  
이 파일에 비인가자가 접근할 수 있으면 다음과 같은 위험이 발생한다.

- 비밀번호 해시 유출
- 오프라인 크래킹 공격 시도
- 비밀번호 정책 우회
- 계정 인증 정보 변조
- 관리자 계정 탈취 위험 증가

이 항목의 목적은 **암호 해시 파일을 관리자만 제어하도록 제한하는 것**이다.

---

## 3. 점검 목적

암호 해시 저장 파일을 관리자만 제어하도록 해 비인가자의 접근·변조를 막기 위함

---

## 4. 주요 확인 파일

### Linux/Solaris
- `/etc/shadow`

### 플랫폼별 추가 확인 파일
- `/etc/security/passwd`
  - AIX 계열 비밀번호 저장 파일
- `/tcb/files/auth`
  - HP-UX Trusted Mode 관련 경로

---

## 5. 확인 명령어

```bash
ls -l /etc/shadow
ls -l /etc/security/passwd
ls -ld /tcb/files/auth
