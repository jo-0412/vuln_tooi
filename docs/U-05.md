# U-05 root 이외의 UID가 0 금지

## 1. 개요

- **항목 코드**: U-05
- **항목명**: root 이외의 UID가 0 금지
- **중요도**: 상
- **분류**: UNIX > 계정 관리

이 항목은 root 외에 UID가 0인 계정이 존재하는지 점검하는 항목이다.

---

## 2. 왜 이 항목이 필요한가

UNIX/Linux 시스템에서 **UID 0은 계정명과 무관하게 최고 관리자 권한**을 의미한다.  
즉 계정명이 `root`가 아니더라도 UID가 0이면 root와 동일한 권한을 가진다.

이 경우 다음과 같은 문제가 발생할 수 있다.

- 숨은 관리자 계정 운영
- 권한 위장 및 우회 접속
- 감사 로그 추적 곤란
- 계정 책임 소재 불명확
- 보안 점검 시 관리자 계정 누락 가능성

이 항목의 목적은 **root 외 UID 0 계정을 식별하여 관리자 권한 중복을 방지하는 것**이다.

---

## 3. 점검 목적

root와 동일한 UID(0)를 가진 숨은 관리자 계정이 없는지 확인하기 위함

---

## 4. 주요 확인 파일

- `/etc/passwd`

---

## 5. 확인 명령어

```bash
awk -F: '$3==0 {print}' /etc/passwd
cat /etc/passwd
