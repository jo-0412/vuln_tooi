# U-54 Disable Unencrypted FTP Service

## 1. Overview

- **Item Code**: U-54
- **Item Name**: Disable Unencrypted FTP Service
- **Severity**: High
- **Category**: UNIX > Service Management

This item checks whether unencrypted FTP services are disabled.

---

## 2. Purpose

The purpose of this check is to prevent account credentials and transferred files
from being exposed through plaintext FTP communication.

Plain FTP does not encrypt usernames, passwords, or transferred data.
If FTP traffic is captured on the network, credentials and sensitive files may be exposed.

---

## 3. Files to Check

- `/etc/inetd.conf`
- `/etc/xinetd.d/ftp`
- FTP daemon configuration files, such as:
  - `/etc/vsftpd.conf`
  - `/etc/vsftpd/vsftpd.conf`
  - `/etc/proftpd/proftpd.conf`
  - `/etc/pure-ftpd/pure-ftpd.conf`

---

## 4. Commands to Check

```bash
grep ftp /etc/inetd.conf
grep -i disable /etc/xinetd.d/ftp 2>/dev/null
systemctl list-units --type=service | grep -E 'vsftpd|proftpd|ftpd'
systemctl list-units --type=socket | grep -E 'vsftpd|proftpd|ftpd'
